# MDD Mobile App Project

## Techstack

- **MDD-client**: Flutter (Dart)

## Folder Structure

```
MDD-client/
├── android/
├── assets/
├── build/
├── ios/
├── lib/
├── test/
├── web/
└── windows/
```

## Setup and Run

### MDD-client (Flutter)

1. Navigate to `MDD-client/`
2. Install dependencies: `flutter pub get`
3. Run the app: `flutter run`
